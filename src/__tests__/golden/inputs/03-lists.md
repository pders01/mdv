# Lists

Unordered:

- One
- Two
- Three

Ordered:

1. First
2. Second
3. Third

Starting at 5:

5. Five
6. Six
7. Seven

Nested:

- Top
  - Mid
    - Deep
  - Mid 2
- Top 2

Mixed nesting:

1. Ordered top
   - Bullet child
   - Another bullet
     1. Numbered grandchild
     2. Another numbered
2. Ordered second

Loose list (with blank lines):

- Item one paragraph.

  Item one second paragraph.

- Item two.

Task list:

- [ ] Todo
- [x] Done
- [ ] Another todo
  - [x] Nested done

List with code:

- Item with code:

  ```ts
  const x = 1;
  ```

- Plain item.

List with blockquote:

- Item with quote:

  > Quoted text.

- Plain item.
