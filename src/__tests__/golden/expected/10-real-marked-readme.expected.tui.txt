 • ⚡ built for speed                                                           
 • ⬇️ low-level compiler for parsing markdown without caching or blocking for   
 long periods of time                                                           
 • ⚖️ light-weight while implementing all markdown features from the supported  
 flavors & specifications                                                       
 • 🌐 works in a browser, on a server, or from a command line interface (CLI)
