# Blockquotes

Simple:

> A single-line quote.

Multi-line:

> Line one of a longer quote
> that wraps across two lines.

Multi-paragraph:

> First paragraph.
>
> Second paragraph.

Nested:

> Outer quote
>
> > Inner quote
> >
> > > Deeply nested

Quote with content:

> A quote with **bold**, `code`, and a [link](https://example.com).
>
> ```ts
> const inside = "block";
> ```
>
> - List
> - In quote

Lazy continuation:

> Lazy continuation
without quote marker on second line.

GitHub-style alert (note):

> [!NOTE]
> Useful information that users should know.

GitHub-style alert (warning):

> [!WARNING]
> Critical content demanding immediate attention.
